# CSS3

CSS3 library for chameleonui

## Installation

Install with [component](https://github.com/component/component):

    $ component install chameleonui/css3

## API

```css
// style.styl
.class-name
    display: block
    border-radius: 20px
    box-shadow: 0 2px 2px black, 0 0 1px 30px rgba(0,0,0,.1)
```

```css
```

### Contains

* animation
* backface-visibility
* background-size
* border-image
* border-radius
* box-shadow
* filter
* font-smoothing
* opacity
* perspective
* text-overflow
* transform
* transition
